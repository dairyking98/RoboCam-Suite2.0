# Archive — RoboCam-Suite 1.0

This directory contains the complete, unmodified source code of **RoboCam-Suite 1.0**, preserved here for historical reference.

The 1.0 codebase was a Raspberry Pi-specific implementation built around `picamera2`, `RPi.GPIO`, and `tkinter`. It served as the functional baseline and design reference for the RoboCam-Suite 2.0 overhaul.

See `docs/RoboCam-Suite_2.0_Technical_Assessment.md` in the repository root for a full analysis of the 1.0 architecture and the rationale for the 2.0 redesign.
